## Installation

1. Clone this Repository

```sh
  https://github.com/NawiOne/cli-convert-txt-json.git
```

2. Install Package

```sh
  npm install
```

3. Run the App
```
- run locally
```sh
node index.mjs 
```

## Usage

1. example

```sh
  conv E:\Hello.txt -t json -o E:\helo\Hello.json
```

2. option

```sh
  -t <type> json | text
```
```sh
  -o <new directory> E:\helo\Hello.txt
```
